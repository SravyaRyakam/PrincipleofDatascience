The Dataset is the Fragility of Female People.

The given table represents data from 10 female participants with the columns representing:

Height: This category represents a person's vertical measurement.

Weight: This attribute signifies a person's mass or body weight.

Age: This feature denotes the number of years a person has lived.

Grip Strength: This characteristic reflects a person's hand grip strength or power.

Frailty: This element indicates whether an individual exhibits fragility, with values "Y" for yes and "N" for no.
